# Product Context

## Why This Project Exists

Starting a new Next.js project from scratch requires installing and configuring multiple tools, reading documentation across several libraries, and wiring up MCP servers for agentic coding — repetitive work that can be eliminated with a well-maintained template.

## Problems It Solves

- Eliminates boilerplate setup time for every new Next.js project
- Ensures consistent tooling (testing, styling, type safety) across projects
- Gives Claude immediate access to library documentation via MCP servers, enabling accurate agentic code generation from day one

## User Experience Goals

- Clone the public GitHub repo and have a fully configured Next.js environment ready in minutes
- All tool configurations pre-wired so no manual setup steps are needed
- MCP servers available out of the box for agentic coding sessions
- README clearly explains what's included and how to get started

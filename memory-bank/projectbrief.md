# Project Brief

## Project Name
nextjs-template

## Overview
A template GitHub repository for bootstrapping Next.js applications.

## Core Requirements

- Install and configure Next.js (App Router, TypeScript)
- Install and configure Tailwind CSS
- Install and configure Vitest (unit/integration testing)
- Install and configure Playwright (end-to-end testing)
- Connect MCP servers for documentation of: Next.js, Tailwind CSS, Vitest, and Playwright
- Write a README.md explaining how to use the template
- Publish as a public GitHub repository so anyone can clone it

## Goals

- Provide a ready-to-clone GitHub repository so any new Next.js project starts with the full tool chain already wired up
- Enable agentic coding by giving Claude access to up-to-date library documentation via MCP servers

## Scope

- In scope: Next.js (App Router), TypeScript, Tailwind CSS, Vitest, Playwright, MCP server configuration, README.md, public GitHub repository
- Out of scope: application-specific business logic, authentication, databases, deployment config
